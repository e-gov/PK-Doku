# Postkast 2.0 liidestuja dokumentatsioon

Käesolev kaust sisaldab Postkast 2.0 liidestujatele mõeldud tehnilist dokumentatsiooni. Dokumentatsiooni eesmärk on koondada ühte kohta liidestumise üldpõhimõtted, liidestumise eeltingimused, teenusliideste kasutamise lähtealused ning muud praktilised kokkulepped, mida rakendusliidese kirjeldus üksinda ei kata.

Dokumentatsioon on koostatud liidestuja vaatest. Siin ei kirjeldata süsteemi sisemist ülesehitust, teenustevahelisi töövooge ega muid sisemisi teostuslahendusi, mis ei ole välise liidese kasutamiseks vajalikud.

## Oluline märkus

Rakendusliidese tehniline kirjeldus on päringute ja vastuste osas esmane lepinguallikas. Käesolev dokumentatsioon täiendab seda kirjeldust ning koondab tingimused, mida liidestuja peab arvestama enne arenduse, testimise ja kasutuselevõtu alustamist.
